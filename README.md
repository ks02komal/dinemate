# 🍽️ DineMate – Smart Restaurant Booking System

 **DineMate** is a full-stack web application designed to streamline restaurant table reservations and management. It enables users to book tables online while providing administrators with tools to efficiently manage bookings, customers, and overall operations.

---

## 🌟 Live Features

## 👤 User Module
  User registration and login system
  Browse available services
  Online table booking functionality
  View and manage booking details
  
## 🛠️ Admin Module
  Admin dashboard overview
  Manage bookings and reservations
  User management system
  Update booking status and records

## ⚙️ Installation & Setup

1. Clone the repository:
  git clone https://github.com/your-username/dinemate.git

2. Move project to local server directory:
  XAMPP → htdocs

3. Setup database:
  Open phpMyAdmin
  Create database: dinemate
  Import file: database/dinemate.sql

4. Configure database connection:
  Open file: config/db.php

5. Update credentials:
  $host = "localhost";
  $user = "root";
  $password = "";
  $database = "dinemate";

6. Run the project:
http://localhost/dinemate
